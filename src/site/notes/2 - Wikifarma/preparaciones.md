# Microscopios
La herramienta principal en la biología celular, histología y organografía es el microscopio, una herramienta que permite observar estructuras a muchos aumentos. Los microscopios más comunes son los ópticos y electrónicos, pero hay más:
- **Óptico:** Tiene una resolución de unos 240 nm (bastante bueno para tejidos) y permite ver a color.
- **MET:** Tiene una resolución de unos 0,2 nm y se ve a una escala de grises que corresponde a la densidad electrónica. Es útil para ultraestructuras celulares, pero no se pueden ver células vivas ya que requiere un vacío para funcionar
- **Contraste de fases y de Nomarski:** Se usan para ver objetos incoloros como los sedimentos y para estudiar cosas vivas.
- **De fluorescencia:** Usado para ver estructuras marcadas con fluorocromos o autofluorescentes.
- **MEB:** Tiene algo menos de resolución que el MET, pero aporta mayor profundidad.

# Preparaciones
Para preparar una muestra, primero se fija, es decir, se trata químicamente para que conserve sus estructuras tras la muerte. Esto se hace normalmente con formol para el microscopio óptico o glutaraldehído en el electrónico. Luego, se deshidratan las muestras y se sumergen en parafina para el M.O. y en araldita para el ME. Luego se cortan las muestras y en el caso del M.O. se pasa por una batería de tinción

La tinción más común es la hematoxilina-eosina, donde la eosina es un basófilo rosita y la hematoxilina es un acidófilo más bien morado
